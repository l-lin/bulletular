Bulletular

Yet another TODO web application with keyboard shortcuts using AngularJS and AngularUI.

Saved!
Shortcuts:
  • CTRL + Enter: Complete the task
  • CTRL + ALT + Backspace: Remove the task
  • UP / DOWN: Switch task
  • TAB: Increase indent
  • SHIFT + TAB: Decrease indent
  • ALT + Space: Expand/Collapse
